LSPosed

Xposed

是一个框架，基于这个框架的模块有很多。

原理是：修改系统的关键文件，然后当 app 调用系统 API 时，首先经过 Xposed，而这些基于 Xposed 的模块就可以选择性的在 app 调用这些 api 的时候干一些”坏坏”的事情，或者修改返回的结果，这样app 在运行的时候效果就会改变，但app本身并没有被破坏，只是调用系统 api 的时候，Android系统的表现发生了变化，这就是钩子 (hook)。Xposed 就是个强大的钩子框架。

LSPosed

Xposed 不分由说的注入所有 app ，尽管它只需要修改某一个 app ，这对系统环境造成了极大的破坏 于是 出现了。兼容Xposed模块的同时，可以为不同 Xposed模块 选择模块要注入的 app 。这对环境稳定非常有帮助。 同时 Xposed 已经成为历史名词，今后也只需要安装 LSPosed 框架。 在 magisk 中打开 Zygisk 选项，然后安装 LSposed 框架open in new window 。

变砖

状况：安装某个 magisk 模块后，启动时卡在了开机动画。解决方法：

  1. 卸载所有 magisk 模块。命令： ::: adb shell magisk --remove-modules :::
  2. 第三方 recovery 直接删除模块文件。路径为 " /data/adb/modules "。瞪眼法认出刚才安装的模块，删除该文件夹。
